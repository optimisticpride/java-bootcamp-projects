# Java Learning Bootcamp

This repository showcases my progress in learning Java fundamentals through practical exercises and weekly assessments.

## Contents
- **Practicals**: Beginner programs covering input/output, conditionals, and arithmetic.
- **Weekly Assessment**: Theory questions, debugging practice, and applied coding tasks.

## Key Concepts Learned
- Using `Scanner` for user input
- Variables (`int`, `double`, `String`)
- Conditional statements (`if`, `else if`, `else`)
- Arithmetic operators (`+`, `-`, `*`, `/`)
- Debugging and fixing syntax errors
- Writing clean, structured Java programs
